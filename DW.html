ABM DW 10

ABM DW 10 is Double Wheel, Cricket Bowling Machine suitable for beginners or junior level cricketers. More robust and feature rich for 3 axis variation ( Line, Length and Height ). These machines can be operated remotely( *Optional ).

Features

* Swings ( In and Out )

* Spin ( Leg and Off )

* Spot Pitching

* Speed up to 125 kmph

* Adjustment to release Dimple and Tennis Balls
